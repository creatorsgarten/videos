description: |-
  พบกับคุณปั่น ผู้ก่อตั้งและ CTO ของ Paypers แพลตฟอร์มจัดการค่าใช้จ่ายองค์กร ผู้มีความเชี่ยวชาญจาก Thoughtworks และระบบโครงสร้างพื้นฐานระดับ blockchain ที่จะมาแบ่งปันประสบการณ์การนำ Claude Code มาพัฒนาซอฟต์แวร์ในระดับ production คุณปั่นจะเจาะลึกเทคนิคการนำ AI มาใช้ใน workflow การทำงานจริง เช่น การสร้าง skill เพื่อจัดการ bug อัตโนมัติ การสร้างแผนงานให้อ่านง่ายในรูปแบบ HTML และการเปิดให้คนในทีม product สามารถดึง insight จากฐานข้อมูลได้เอง
